Readme:
This project enables a user to enter a stock name, and returns a detailed analysis of the stock using sentiment data (collected from twitter and analyzed in main) as well as historical price data. This application then uses gradient descent to find regressions and predict future prices, thus giving recommendations to buy or sell the stock in the long term/short term.
