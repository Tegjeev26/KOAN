Help

Evaluate Screen: Enter a stock ticker symbol, and an appropriate time period in years (ie 0.5, 1, 1.5, 2). We recomend that you don't enter times beyond 2, as the results will be more accurate with shorter time spans.

Analysis Screen: The user can see four charts analysing either the stock price history or the sentiment surrounding the stock. This data is analysed using either a linear regression or a polynomial regression. These trends are used to make the deductions outlined above the charts. The tool can recommend a buy or sell decision depending on the analysis done with four regressions, and also provides the user insight into the works of the algorithms. To learn more about how the analysis was done, click the more details links.

Sentiment Analysis: This screen shows the sentiment expressed in example tweets about the company that were analysed, and emphasizes the important parts of the stock that were analysed. The page also provides useful explanations of how the tool works.

Price Screen: Displays historical price data about the company in question. Below this, it provides an explanation of how the algorithm analysed price history, and displays an example of the regression calculation.
