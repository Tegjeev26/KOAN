Sentiment Analysis

This approach was used in analysing the tweets corresponding to the stock in question, with the goal of extracting relevant and useful information about the company. Specifically, sentiment analysis is geared towards extracting behavioural or opinonated data, and was applied to tweets discussing the company in question in this case.

This tool made use of the 'bag of words' machine learning approach to sentiment analysis, which seeks to assign to each word a sentiment value representing the probability of that word being using in a positive or negative context. Using those weightings, this tool supplied output about public opinion of the company in question, and factored this behavioural data into the stock analysis.
